## 🎯 格式說明

| 格式 | 用途 | 長度 |
|------|------|------|
| 部落格 | 網站文章 | 1500-3000 字 |
| YouTube | 影片腳本 | 3-10 分鐘 |
| 社群媒體 | IG/FB/X/LI | 依平台 |
| EDM | 電子報 | 500-800 字 |
| 播客 | 音訊節目 | 5-30 分鐘 |
| 簡報 | 演講投影片 | 10-20 頁 |
| 資訊圖 | 視覺化 | 5-10 張 |
| 新聞稿 | 媒體發布 | 500-800 字 |
| 行銷文案 | 廣告/銷售頁 | 依需求 |
| 課程大綱 | 教學設計 | 2-4 小時 |

## 💡 使用方式

### 基本指令
```
內容變形：[貼上內容]
```

### 進階指令
```
內容變形：[貼上內容]
目標格式：YouTube 腳本
語言風格：口語化
長度：5 分鐘
```

### 批次轉換
```
內容變形：[貼上內容]
全部格式：一次產出所有 10 種格式
```
